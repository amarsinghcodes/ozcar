# FIX

## PLAN

- Root cause: the Phase 4 Pi export command still depends on a caller-provided snapshot file, so it materializes artifacts from external disk input instead of from validated audit state already stored on the active Pi branch.
- Repair group 1: add the minimum Pi-backed artifact-state seam by validating and checkpointing the latest audit snapshot into a dedicated Pi custom entry type on the active branch.
- Repair group 2: change `/ozcar-audit-export` to rebuild only from the stored Pi snapshot state for the current audit and fail closed on missing or mismatched stored state.
- Repair group 3: update the Phase 4 tests and the small help/prompt/doc surfaces that still describe the old file-based export flow.

## CHANGES

- Replaced the file-based Phase 4 export seam with Pi-backed stored snapshot state:
  - added `.pi/extensions/ozcar/state/audit-artifact-snapshot.ts` to validate, align, checkpoint, and restore the latest `AuditArtifactSnapshot` from a dedicated Pi custom entry on the active branch
  - added `.pi/extensions/ozcar/tools/audit-artifact-snapshot.ts`, which registers `ozcar_store_audit_snapshot` as the minimum repo-owned seam for storing validated Phase 4 snapshot state inside Pi before export
  - changed `.pi/extensions/ozcar/commands/export.ts` so `/ozcar-audit-export` now takes no file argument, reads only the latest stored branch snapshot for the current audit, and fails closed on missing or mismatched stored state
- Updated the extension surface and docs that still described the old file-input flow:
  - `.pi/extensions/ozcar/index.ts` now registers the new tool and advertises the stored-state export flow in `/ozcar` help
  - `README.md`, `.pi/prompts/ozcar-audit.md`, and `.pi/skills/ozcar-audit/SKILL.md` now describe checkpointing the validated snapshot into Pi state first and then running `/ozcar-audit-export`
- Replaced the export-command tests with Pi-backed-state coverage in `tests/pi-extension.export.test.ts` and extended `tests/pi-extension.test.ts` to assert the new tool registration and help text.
