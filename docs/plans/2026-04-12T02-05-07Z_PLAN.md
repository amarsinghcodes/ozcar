`PLAN.md` remains canonical, and the next active slice is now Phase 4: wire the runtime into the full operational CLI on top of the completed Phase 1, Phase 2, and Phase 3 seams. - Phase 4 command completion under `src/commands/{run,resume,status,replay,rebuild,doctor}.ts`

- tests and validation for resume, replay, rebuild, doctor, and the durable filesystem seams they depend on
